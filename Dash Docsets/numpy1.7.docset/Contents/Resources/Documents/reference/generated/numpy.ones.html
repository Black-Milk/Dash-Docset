numpy.ones¶

numpy.ones(shape, dtype=None, order='C', maskna=False)¶

Return a new array of given shape and type, filled with ones.

Please refer to the documentation for zeros for further details.

See also

zeros, ones_like

Examples

>>> np.ones(5)
array([ 1.,  1.,  1.,  1.,  1.])

>>> np.ones((5,), dtype=np.int)
array([1, 1, 1, 1, 1])

>>> np.ones((2, 1))
array([[ 1.],
       [ 1.]])

>>> s = (2,2)
>>> np.ones(s)
array([[ 1.,  1.],
       [ 1.,  1.]])
