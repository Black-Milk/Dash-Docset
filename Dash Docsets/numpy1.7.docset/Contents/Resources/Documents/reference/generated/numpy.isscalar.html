numpy.isscalar¶

numpy.isscalar(num)¶

Returns True if the type of num is a scalar type.

Parameters :

num : any

Input argument, can be of any type and shape.
Returns :

val : bool

True if num is a scalar type, False if it is not.

Examples

>>> np.isscalar(3.1)
True
>>> np.isscalar([3.1])
False
>>> np.isscalar(False)
True
