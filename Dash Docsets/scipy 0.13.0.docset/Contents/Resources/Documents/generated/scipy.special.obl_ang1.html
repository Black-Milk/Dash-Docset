scipy.special.obl_ang1¶

scipy.special.obl_ang1(x1, x2, x3, x4[, out1, out2]) = <ufunc 'obl_ang1'>¶

(s,sp)=obl_ang1(m,n,c,x) computes the oblate sheroidal angular function of the first kind and its derivative (with respect to x) for mode parameters m>=0 and n>=m, spheroidal parameter c and |x| < 1.0.
