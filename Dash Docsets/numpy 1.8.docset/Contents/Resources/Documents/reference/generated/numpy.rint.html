numpy.rint¶

numpy.rint(x[, out]) = <ufunc 'rint'>¶

Round elements of the array to the nearest integer.

Parameters :

x : array_like

Input array.
Returns :

out : {ndarray, scalar}

Output array is same shape and type as x.

See also

ceil, floor, trunc

Examples

>>> a = np.array([-1.7, -1.5, -0.2, 0.2, 1.5, 1.7, 2.0])
>>> np.rint(a)
array([-2., -2., -0.,  0.,  2.,  2.,  2.])
