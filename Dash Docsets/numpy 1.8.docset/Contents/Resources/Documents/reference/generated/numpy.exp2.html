numpy.exp2¶

numpy.exp2(x[, out]) = <ufunc 'exp2'>¶

Calculate 2**p for all p in the input array.

Parameters :

x : array_like

Input values.

out : ndarray, optional

Array to insert results into.
Returns :

out : ndarray

Element-wise 2 to the power x.

See also

power

Notes

New in version 1.3.0.

Examples

>>> np.exp2([2, 3])
array([ 4.,  8.])
