numpy.ma.MaskedArray.shrink_mask¶

MaskedArray.shrink_mask()[source]¶

Reduce a mask to nomask when possible.

Parameters :None
Returns :None

Examples

>>> x = np.ma.array([[1,2 ], [3, 4]], mask=[0]*4)
>>> x.mask
array([[False, False],
       [False, False]], dtype=bool)
>>> x.shrink_mask()
>>> x.mask
False
