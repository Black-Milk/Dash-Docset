numpy.ma.compress_rows¶

numpy.ma.compress_rows(a)[source]¶

Suppress whole rows of a 2-D array that contain masked values.

This is equivalent to np.ma.extras.compress_rowcols(a, 0), see extras.compress_rowcols for details.
