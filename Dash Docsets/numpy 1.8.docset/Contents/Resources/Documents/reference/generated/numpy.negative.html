numpy.negative¶

numpy.negative(x[, out]) = <ufunc 'negative'>¶

Returns an array with the negative of each element of the original array.

Parameters :

x : array_like or scalar

Input array.
Returns :

y : ndarray or scalar

Returned array or scalar: y = -x.

Examples

>>> np.negative([1.,-1.])
array([-1.,  1.])
