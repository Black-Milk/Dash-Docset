8.9.5. sklearn.feature_selection.SelectFwe¶

class sklearn.feature_selection.SelectFwe(score_func=<function f_classif at 0x108932c80>, alpha=0.05)¶

Filter: Select the p-values corresponding to Family-wise error rate

Parameters :

score_func : callable

Function taking two arrays X and y, and returning a pair of arrays (scores, pvalues).

alpha : float, optional

The highest uncorrected p-value for features to keep.

Attributes

pvalues_ array-like, shape=(n_features,) p-values of feature scores.
