scipy.special.itmodstruve0¶

scipy.special.itmodstruve0(x[, out]) = <ufunc 'itmodstruve0'>¶

y=itmodstruve0(x) returns the integral of the modified Struve function of order 0 from 0 to x: integral(L0(t), t=0..x).
